## **ขั้นที่ 2️⃣: Vercel Deploy**

### **2.1 Vercel Login**

```
1. ไปที่: vercel.com
2. Click "Sign up"
3. "Continue with GitHub"
4. GitHub ยืนยัน → "Authorize"
```

### **2.2 Deploy Repository**

```
1. Vercel Dashboard
2. Click "New Project"
3. หา Repository "THE_VIEW_Loyalty"
4. Click "Select"
5. Click "Deploy"
```

### **2.3 รอ Deploy เสร็จ**

```
Vercel กำลังทำงาน:
  → 30-60 วินาที
  → เห็น "Congratulations!"
  → ได้ URL:
    https://the-view-loyalty.vercel.app ✓
```
