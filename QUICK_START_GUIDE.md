# 🚀 Quick Start Guide - Loyalty App with Registration + PDPA

## ⚡ ขั้นตอน 5 นาที - เริ่มใช้เลย

### 1️⃣ ดาวน์โหลด + เปิด
```
1. ดาวน์โหลด: loyalty_with_registration.html
2. Double-click → เบราว์เซอร์เปิดโดยอัตโนมัติ ✓
3. อนุญาตกล้องครั้งแรก → เสร็จ!
```

### 2️⃣ เพิ่มสมาชิกใหม่ (ครั้งแรก)
```
ปุ่ม "➕ เพิ่มสมาชิกใหม่"
  ↓
- ชื่อ-นามสกุล: "สมชาย สุขไทย"
- เบอร์โทร: "081-234-5678"
- เพศ: "ชาย"
- วันเกิด: "15 มีนาคม 2532" (ระบบคำนวณอายุเอง)
- ✓ ติ๊ก PDPA Consent checkbox
  ↓
"ลงทะเบียน" → ✓ เสร็จ!
```

### 3️⃣ สแกนและให้คะแนน
```
ปุ่ม "📱 สแกน QR Code"
  ↓
ชี้กล้องไปยัง QR ลูกค้า
  ↓
ระบบจะแสดง Detail ลูกค้า → เลือก "+50" คะแนน
  ↓
✓ เพิ่มคะแนนเสร็จ!
```

### 4️⃣ ดูสถิติ (Analytics)
```
ปุ่ม "📊 วิเคราะห์ข้อมูล"
  ↓
แสดง:
- จำนวนลูกค้าทั้งหมด
- แบ่งตามเพศ (ชาย/หญิง/ไม่ระบุ)
- แบ่งตามช่วงอายุ
- แบ่งตามระดับ Tier
```

### 5️⃣ Export ไป Google Sheets (สำหรับวิเคราะห์)
```
ปุ่ม "💾 Export CSV"
  ↓
ได้ไฟล์: loyalty_customers_2025-01-15.csv
  ↓
วิธีต่อไป: ↓
```

## 📊 วิธี Export ไป Google Sheets

### ขั้นที่ 1: Export CSV
1. เปิดแอป Loyalty
2. กดปุ่ม "💾 Export CSV"
3. ได้ไฟล์ชื่อ `loyalty_customers_[วันที่].csv`

### ขั้นที่ 2: เปิด Google Sheets
1. เข้า [google.com/sheets](https://sheets.google.com)
2. สร้าง Spreadsheet ใหม่

### ขั้นที่ 3: นำเข้า CSV
1. **File** → **Import** → เลือก CSV ที่บันทึก
2. ตัวเลือก:
   - Replace spreadsheet: ✓ (ถ้าอยากแทนที่)
   - Append to sheet: ✓ (ถ้าอยากเพิ่ม)
3. **Import** → ✓ เสร็จ!

### ขั้นที่ 4: วิเคราะห์ใน Sheets

#### วิธี 1: Pivot Table
```
ข้อมูล → Pivot Table
  ↓
Rows: เพศ
Values: นับจำนวน
  ↓
ได้ตาราแสดง: ชาย/หญิง/ไม่ระบุ กี่คน
```

#### วิธี 2: Filter
```
เลือกแถว Header → Data → Create a filter
  ↓
- คลิกลูกศร "เพศ" → filter เฉพาะ "หญิง"
- คลิกลูกศร "อายุ" → filter เฉพาะ "25-34"
  ↓
ได้กลุ่มเป้าหมาย: หญิง 25-34 ปี → กี่คน?
```

#### วิธี 3: Chart
```
เลือก Column A:C (เพศ, ช่วงอายุ, จำนวน)
  ↓
Insert → Chart
  ↓
Chart type: Pie Chart / Bar Chart
  ↓
ได้กราฟแสดงภาพ demographic
```

## 🆘 Troubleshooting

### ❓ สแกน QR ไม่ได้
**A:** 
1. เช็ค QR code ชัดหรือไม่
2. ป้อน ID ด้วยตนเอง (ถ้าสแกนไม่ได้)
3. ตรวจสอบว่ากล้องทำงานปกติ

### ❓ ข้อมูลหายไป
**A:** 
1. ตรวจสอบว่าลบเบราว์เซอร์หรือไม่
2. ลองเปิด Private/Incognito mode ได้หรือไม่?
3. Export CSV ก่อนลบ

### ❓ Export CSV แล้ว Sheets ไม่รู้สึก
**A:**
1. ไฟล์ CSV ต้องเป็น UTF-8 (ตัวอักษรไทย)
2. ลองเปิดด้วย Google Sheets โดยตรง
3. ลองอัพโหลดผ่าน "File → Import"

### ❓ รหัสลูกค้าซ้ำ
**A:**
1. ไม่ให้กรอกรหัส → ปล่อยให้ระบบสร้าง
2. ระบบจะสร้าง ID ไม่ซ้ำ "C" + timestamp
